Elements¶

Platforms: Linux, Mac, Windows

Basic Usage¶

class periodic.element(symbol)¶

Takes a periodic data (ex: ‘Ag’, ‘12’, 1, 15.999) as input. It relies on the finder function.

>>> import periodic
>>> osmium=periodic.element('Os')
>>> osmium.name
'osmium'
>>> osmium.symbol
'Os'
>>> osmium.atomic
76
>>> osmium.charge
(3, 4)
>>> osmium.type
'metal'

chem also includes loose atomic mass variables

>>> from periodic.mass import *
>>> H
1.0079
>>> H+Cl
36.4609
>>> H+CL
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
NameError: name 'CL' is not defined

Note

Notice the NameError when we tried to use CL. Remember that all the elements have the first letter capitalized only.

New in version 2.0: This whole module was re-written, with the same data.
